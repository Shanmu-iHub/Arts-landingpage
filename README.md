# Dr. SNS Rajalakshmi College of Arts and Science - Landing Page

Welcome to the **Arts-landingpage** repository! This project contains the source code for the official landing page of Dr. SNS Rajalakshmi College of Arts and Science. 

## 🌐 Overview

This modern, fully responsive landing page is designed to showcase the institution's offerings, academic excellence, sprawling innovation ecosystem, and robust placement records. It features a premium dark-themed design with vibrant brand colors and dynamic micro-interactions.

## ✨ Key Features

- **Hero Section:** Engaging introduction highlighting Design Thinking, AI startups vision, and quick statistics.
- **Academic Programs (Arts Layout):** A 29-item grid categorizing UG and PG Arts & Science programs (Commerce, B.Sc Computer Science, Specialized PG).
- **Placement Excellence:** Real-time counter animations (1642+ Students Placed, 24 LPA Highest Package, 85% Placement Rate).
- **Research & Innovation:** Stats tracking Patents, Research Articles, Startups, Funded Projects, and Industry MOUs.
- **Global Certifications:** A dense 5x2 grid showcasing 13 IT Core certifications (Azure, AWS, Databricks) and 4 Non-IT Core certifications (SAP FICO, NSE, Tally, CPA).
- **SPINE Experience:** Interactive cards (4x2 grid) detailing the 5-level activity center with custom-colored icons.
- **Innovation Ecosystem:** Six specialized innovation labs highlighting Bot Lab, IBI Liquid Studio, and I-Tech Labs.

## 🛠️ Technology Stack

- **HTML5:** Semantic architecture
- **Tailwind CSS:** Utility-first framework for rapid UI development and fully responsive layouts.
- **Vanilla JavaScript:** For interactive tabs, sticky headers, mobile menus, and dynamic number counters.
- **Font Awesome:** Scalable vector icons mapping neatly to the brand's primary color palette.

## 🚀 Getting Started

To view the landing page locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Shanmu-iHub/Arts-landingpage.git
   ```
2. Open `index.html` in your favorite web browser. No complex build tools or servers are required!

## 🎨 Design System

**Primary Color Palette:**
Brand colors feature an energetic, high-contrast **Green** (`#B0CB1F`) built atop deep **Grays/Blacks** (`#111827`, `#1F2937`) to highlight essential elements like duration badges and call-to-action buttons.
